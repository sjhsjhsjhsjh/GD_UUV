# 运动规则
敌方固定在 x=2000,z=150 的位置，仅允许随机做左右运动，不允许改变X和Z轴坐标

# 坐标系定义
使用空间笛卡尔坐标系，但是说“向前”“向后”这种描述性语言是指：在UUV的初始位置，向enemy的初始位置看去的方向为参考方向。即，x轴负方向为向前，因为enemy在x=2000但uuv在x>8000。这个所谓前后不影响实际坐标。

# 知识
1. 强化学习，至少 PPO 不需要将 reward 归一化，因为价值网络会自动拟合这个区间。当然，不能太大。
2. 网络输入，各个特征最好归一化。网络训练有这样的偏好：
   1. 关键数据和 reward 的关系，最好是正相关。
   2. 不要将数据全部弄成负的。如果是Relu不好训练。
   3. [-1,1],[0,1]之间的分布都是可以的
   4. 遵循“零均值化”和“尺度对齐”
